## Properties
###### [`| MUI Reference |`](https://mui.com/material-ui/api/snackbar/)
<br/>

* `children:` element
* `classes:` object
* `type:` 'success' | 'info' | 'warning' |'error'
    * default: 'success'
* `action:` ReactNode
* `anchorOrigin:` { horizontal: 'center' | 'left' | 'right', vertical: 'bottom' | 'top'}
* `showAction:` bool
    * default: false
* `autoHideDuration:` number\*
* `ClickAwayListenerProps:` object
* `disableWindowBlurListener:` bool\*
* `key:` any
* `message:` ReactNode
* `onClose:` func
* `open:` bool\*
* `resumeHideDuration:` number
* `sx:` Array<func | object | bool> | func | object
* `transitionDuration:` number | {appear?: number, enter?: number, exit?: number}

## Dependencies
* &lt;[ClickAwayListener](https://mui.com/base/api/click-away-listener/)/&gt;

## Implementation Uncertainties
* May be easier to implement this one using the MUI component, as it has certain complexities (such as timeouts) which will be more work to implement from scratch
