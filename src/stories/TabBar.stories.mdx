## Implementation Uncertainties
* the `action` prop function is run when it mounts, it was taken from MUI. Not sure about implementation and if it is necessary. 
* not sure if we need the `aria-label` and `aria-labelled-by`
* On MUI, there are a lot of properties for scrolling, not sure how the design team thinks about the tab bar being able to scroll or not
